# ジャイロスコープを確認する

## 前面を定義し、変更する可能性を想定する

前面を定義することが最も簡単です。希望の前面でいいですが、変更が必要になる可能性を想定しておいてください。モジュールの相対的な位置はジャイロスコープによって決まります。ジャイロスコープが誤った向きで設置されており変更できない場合、既存のサンプルや YAGSL には含まれていないコードで補正する必要があります。

{% hint style="warning" %}
前面はジャイロスコープの `0` で決まります！
{% endhint %}

## ジャイロスコープを校正する...

NavX やその他のジャイロスコープを使用している場合は、特に大会前に少なくとも 1 回は校正することを強くお勧めします。NavX は工場（ハワイ）の湿度に合わせて校正されており、ショップに届いたら必ず再校正してください。特定のジャイロスコープについての詳細なドキュメントはこちらにあります。

{% content-ref url="../devices/gyroscope.md" %}
[gyroscope.md](../devices/gyroscope.md)
{% endcontent-ref %}

## ジャイロスコープを反転させる理由は？

ジャイロスコープは、反時計回りに移動してもヨーが増加しない場合**のみ**反転させてください！
