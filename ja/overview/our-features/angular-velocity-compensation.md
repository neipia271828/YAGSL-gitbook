# 角速度補正

YAGSL は Jack-in-the-bot が先駆けた角速度補正の手法を採用しています。有効にするには `SwerveDrive.setAngularVelocityCompensation(true, true, 0.1);` を使用します。第 1 引数はテレオップでの有効状態、第 2 引数は自律制御での有効状態、第 3 引数は使用する係数です。

初めて有効にしたときにスキューが大幅に悪化した場合は、値を反転させてみてください。

1. 回転しながら直線移動してチューニングします。
   * 右スティックの角速度制御でテストするのが最適です。
2. スキューが視覚的に満足できるまで値を変更します。
3. さまざまな並進・回転の大きさでチューニングが機能することを確認します。
4. テレオップでスキューが改善されれば、自律制御でも改善される可能性があります。
